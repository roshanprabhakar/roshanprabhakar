QuizGrader

Quiz Grader is a simple yet effective tool used to help teachers grade tests and quizes. The software takes electronic copies all tests then allows the test to go through the test and grade each question instead of each test. A teacher using the software would grade all responses to one question, then move on to the next question. According to multiple teachers, this allows teachers to preserve context when grading questions.

Flower Classifier

Flower Classifier trains a perceptron to classify a flower as a Setosa, Virginica, or Versicolor. Trained with the Iris dataset, current classification yields 99% accuracy. Program also generates an error map. See map implementation to read map. The regression model can be viewed at this link. Currently, x: sepal-length, y: sepal-width, but higher dimension classification is possible, but not visually presentable. first-fifth power regression models are graphed, but higher orders are possible.

Facial HAAC Image Compression Project

An image compression program for face images. The program identifies keystone features of a face in an image necessary for human recognition. These features are preserved through scaling, while the rest of the image is compressed through a much larger scaling ratio.
